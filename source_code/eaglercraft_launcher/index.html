Eaglercraft launcher

Eaglercraft is a way to play Minecraft java on your chromebook through a web browser for free. However, you cannot play any minecraft server. you must play a server that has the mod "eaglerXbungee", such as archmc (mc.arch.lol).


Version 1.8.8

Version 1.8.8 of eaglercraft is based off of minecraft 1.8.8 (stable)

Version 1.12.2

Version 1.12.2 is based on minecraft 1.12.2 (unstable)
